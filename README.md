# ⚙️ Cách cài đặt

- Tải và cài đặt [XAMPP](https://www.apachefriends.org/download.html)  
- Sao chép thư mục dự án này vào đường dẫn **C:/xampp/htdocs** (vị trí mặc định khi cài XAMPP)  
- Mở ứng dụng XAMPP và khởi động hai module **Apache** và **MySQL**  
- Mở [phpMyAdmin](http://localhost/phpmyadmin/) hoặc nhấn nút **Admin** trong module **MySQL** để truy cập nhanh  
- Tạo cơ sở dữ liệu mới có tên **db_sucre_web** và chọn **collation** là **utf8mb4_unicode_ci**  
- Import file **db_sucre_bakery.sql**  vào cơ sở dữ liệu vừa tạo  
- Sau khi hoàn tất, truy cập **giao diện người dùng (User)** tại [http://localhost/sucre-bakery/](http://localhost/sucre-bakery/)  
- Để truy cập **giao diện quản trị (Admin)**, vào [http://localhost/sucre-bakery/admin.php](http://localhost/sucre-bakery/admin.php)
